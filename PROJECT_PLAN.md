# NGFit Pro - Roadmap

## 1. Клиентский портал (Client Portal)
Разделение ролей в приложении.
- **Роли:**
  - `Admin/Trainer`: Полный доступ (текущий функционал).
  - `Client (Personal)`: Бесплатный доступ. Видит статистику, может вести назначенные тренировки.
  - `Client (Online)`: Платный доступ. Видит платные программы.
- **Функции:**
  - Личный кабинет клиента (статистика, календарь).
  - Дневник тренировок (ввод весов, повторений, комментариев).

## 2. Шаблоны тренировок (Templates)
- Конструктор программ из базы упражнений.
- Назначение шаблона клиенту на дату или период.

## 3. Финансы и Монетизация
- **Оплата:** Интеграция Telegram Stars или ручное подтверждение (скриншоты).
- **Подписки:** Авто-проверка срока действия абонемента.
- **Магазин:** Витрина дополнительных услуг/товаров.

## 4. AI Ассистент (Perplexity Integration)
- **Функция:** Чат-бот внутри приложения.
- **Возможности:** 
  - Ответы на вопросы по питанию (КБЖУ).
  - Консультации по технике упражнений.
  - Помощь тренеру в составлении программ (Drafting).

## 5. База знаний
- Раздел с видео/статьями для клиентов (доступен по подписке или всем).
